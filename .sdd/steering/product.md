# Steering — Produto (fintech-pos-api)

> Contexto persistente que o agente carrega em todas as fases. Preencha antes da primeira spec.

## Visão
API REST de Point-of-Sale para uma fintech, responsável por autorizar transações,
emitir comprovantes e conciliar com o adquirente.

## Personas
- Lojista (operador do PDV)
- Cliente final (pagador)
- Backoffice (conciliação)

## Não-objetivos
- Frontend do PDV (fora do escopo desta API)
- Integração com bandeiras diretamente (vai via adquirente)
