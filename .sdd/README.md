# .sdd/ — Spec-Driven Development workspace

Este diretório é gerenciado pelo `ai-sdd`. Cada feature ganha uma subpasta:

```
.sdd/<feature>/
├── spec.md       # Requisitos (EARS)
├── design.md     # Arquitetura e contratos
└── tasks.md      # Tarefas executáveis
```

**Não edite à mão** durante uma fase ativa; use os slash commands.
